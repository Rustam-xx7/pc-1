# pc-1

My personal sandbox for learning and practicing different technologies. This isn't a single project — it's a growing collection of exercises, mini-projects, and experiments I work through while picking up new languages, frameworks, and concepts.

Think of it as a coding journal. Code quality and completeness vary by folder since some things here are day-one experiments and others are more polished practice projects.

## What's inside

Click any folder name to jump straight to it.

| Folder | Description |
|---|---|

| [`DSA`](./DSA) | Data Structures & Algorithms practice — arrays, linked lists, trees, sorting, searching, problem-solving |
| [`backend`](./backend) | Backend development practice — server logic, APIs, and related concepts |
| [`backend-ex`](./backend-ex) | Additional backend exercises and smaller experiments |

- To have a reference I can come back to when I forget how I solved something
